## Install

- Install packages
    ```
    sudo apt-get install picom blueman xorg-dev arandr i3lock-fancy rofi
    ```
- [Install nvim](https://github.com/3nd3r1/init.lua)
- Install zoxide
    ```
    curl -sS https://raw.githubusercontent.com/ajeetdsouza/zoxide/main/install.sh | bash
    ```
